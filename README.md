# Zig TCP/IP

A simple tcp server written in Zig.

## Requirements

- Zig 0.15

## Tasks

### Core

- [x] Bind to a port
- [x] Respond with 200
- [x] Extract URL path
- [x] Respond with body
- [ ] Read header
- [ ] Concurrent connection
- [ ] Return a file
- [ ] Read request body

### HTTP compression

- [ ] Compression headers
- [ ] Multiple compression schemes
- [ ] Gzip compression

### Persistent connections

- [ ] Persistent connections
- [ ] Concurrent persistent connections
